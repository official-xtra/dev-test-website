## Cara menjalankan di local PC

1. Clone lalu install package :

```bash
npm install
# or
yarn add
```

2. Run development server:

```bash
npm run dev
# or
yarn dev
```

3. Buka [http://localhost:3000](http://localhost:3000) di browser.
